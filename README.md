# Гласов помощник

Voice-to-text на български — личен Wispr Flow.

## Deployment на Vercel (3 стъпки)

### Стъпка 1 — качи в GitHub
1. Създай нов repository на github.com (може да е private)
2. Качи всички файлове от тази папка

### Стъпка 2 — свържи с Vercel
1. Отиди на vercel.com и влез с GitHub акаунт
2. "Add New Project" → избери твоя repository
3. Натисни "Deploy" — Vercel автоматично разпознава Next.js

### Стъпка 3 — добави на iPhone
1. Отвори твоя URL (напр. glasov.vercel.app) в Safari
2. Бутон "Сподели" → "Добави към начален екран"
3. Готово — работи като истинско приложение!

## Как работи
- Записва гласа ти
- Праща аудиото към OpenAI Whisper за транскрипция на български
- Claude оправя пунктуацията и главните букви
- Копираш резултата с един бутон

## API ключове
- **OpenAI**: нужен е за Whisper транскрипция (~$0.006/мин)
- Ключът се пази само в localStorage на твоето устройство
